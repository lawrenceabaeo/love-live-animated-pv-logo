This pure css3 mini-project copies the animated logo that appears in some Love Live anime Promotional Videos. An example of one of those videos can be viewed here: Click here for example video

Why did I make this?

I wanted to see how far I could get in copying the noted video’s opening logo animation using only CSS3 animations. I think I got 90% - I could not copy the particle effects that happen when the English letters turn to Japanese, and including a cloud jpeg sapped performance. For the particles, I substituted an animation that had the 'spirit' of what occurred in the original.

What technology skills were learned/retrieved?

CSS3 animations

What are this project’s limitations?

  • August 17, 2015: Currently only Chrome/Safari smoothly handle these CSS3 animations. Internet Explorer and Firefox on the same machines are choppy.
  • The animation does NOT scale and is NOT mobile responsive. Use the browser’s zoom function to achieve additional zooming.
  • The project is NOT compatible with older browsers.
  • There is no cloud background. I tried to put one in, but chrome on an older laptop couldn't handle it, even when I compressed the picture.
  • The fonts are approximate because I do not want to pay to use the exact commercial fonts.

Tech references

  • Koruri font (for the japanese words)
  • Saxmono font(for the word ‘School idol project’)
